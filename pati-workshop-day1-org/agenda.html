3 실행

  • 메뉴
    • ctrl(cmd) E => edit mode
    • ctrl(cmd) 1 => object
    • ctrl(cmd) 2 => message
    • ctrl(cmd) 3 => number
    • ctrl(cmd) shift B => bang
    • ctrl(cmd) shift T => toggle
    • array
