* 실행
  - 메뉴
    - ctrl(cmd) E => edit mode
    - ctrl(cmd) 1 => object
    - ctrl(cmd) 2 => message
    - ctrl(cmd) 3 => number
    - ctrl(cmd) shift B => bang
    - ctrl(cmd) shift T => toggle
    - array

* 시작
  - "hi"
  - [bang ], [tgl ], [print ] => post window
  - [float ], [f ], [+ ] => counter
  - [- ], [* ], [/ ]
  - [== ], [!= ], [> ], [< ], [>= ], [<= ]

* 계산의 순서
  - cold inlet/hot inlet
  - [trigger ], [t ]

* 퀴즈
  - 1부터 10까지의 합을 구하는 프로그램
  - 1부터 10까지의 곱을 구하는 프로그램
